A segmented switch displaying two options.

Renders a segmented switch displaying optionFirst and optionSecond. Modify via custom passed modifier.

Example:

var selectedIndex by remember { mutableStateOf<Int?>(null) }
SegmentedSwitch(
  optionFirst = { Text("Off") },
  optionSecond = { Text("On") },
  selectedIndex = selectedIndex,
  onOptionClick = { index -> selectedIndex = index },
  label = { Text("Feature") },
)
